# Ultraquiz

get stats data from rubygems.org

## Installation

Add this line to your application's Gemfile:

    gem 'ultraquiz'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ultraquiz

## Usage

```bundle exec ultraquiz fetchcsv```
